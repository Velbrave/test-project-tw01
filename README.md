# Getting Started with Create React App

This project was bootstrapped with
[Create React App](https://github.com/facebook/create-react-app).

Проект "Нотатки"

Цей проект написаний на React.js з використанням бази даних Firebase. Тут можна
зберігати свої нотатки, редагувати їх або видаляти. Також налаштований пошук по
перших літерах потрібної нотатки. В хедері знаходиться навігація та пошук.
Навігація складається з двох кнопок при кліку на які переходиш на сторінку
додавання нотатки, або редагування нотатки. При кліку на кнопку видалення
з"являється модальне вікно і користувач повинен вибрати чі дійсно він хоче
видалити цю нотатку, чі нажав кнопку помилково і може повернутися до
редагування. В верстці використана стратегія Mobile First. Також у цьому проекті
опрацьовано більшість помилок які можуть виникнути,щоб захистити наш додадок.
